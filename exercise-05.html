All sampling distributions appear to be normally distributed. The zombies_killed variable and years_of_education variable are normally distributed, even though they were not normally distributed for the single sample. This is due to the phenomenon of Central Limit Theorem, that if enough large samples are taken from a population, even though the variable may not have a normal distribution within the population, its sampling distribution will be normal.

Step 10 - Bootstrapping - I followed the template for bootstrapping we learned in class and added another for() loop outside of the for() loop for the bootstrapping so that the bootstrapping was done for each variable.

n_boot <- 1000
vars <- c("height", "weight", "age", "zombies_killed", "years_of_education")
ci_results <- list()
for(v in vars) {
  boot <- vector()
  s2 <- z[[v]]
  n <- length(s2)

  for(i in 1:n_boot) {
    boot[[i]] <- mean(sample(s2, n, replace = TRUE))
  }
  ci_results[[v]] <- quantile(boot, probs = c(0.025, 0.975))
}

ci_results

The confidence intervals for the bootstrapped data are much narrower than the confidence intervals from the sampling distribution. This means that bootstrapping is a more accurate way of developing confidence intervals.
